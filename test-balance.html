🔑 API Key 余额查询功能

已成功为单个Key检测组件添加余额查询功能，以下是主要特性：

✅ 余额查询开关

在高级设置中添加了余额查询开关，用户可以选择是否查询账户余额

🎯 智能显示

余额信息以突出的绿色卡片形式显示，格式化为美元金额（保留两位小数）

🔧 技术实现

修改了 AIKeyValidator.validateKey 方法，添加 checkBalance 参数

validateKey(key, provider, format, checkBalance, secretKey)

🆕 硅基流动余额查询

新增硅基流动专用验证方法，支持完整的用户信息和多维度余额查询

API端点: https://api.siliconflow.cn/v1/user/info 认证方式: Bearer Token 返回信息: 总余额、可用余额、充值余额、用户详情等 响应格式: 标准包装格式 {code, message, status, data}

💰 多维度余额显示

硅基流动支持显示详细的余额分解信息

  • 总余额: 账户总可用金额
  • 可用余额: 当前可使用金额
  • 充值余额: 充值获得的金额
  • 账户状态: 正常/冻结等状态
  • 管理员权限: 是否具有管理权限

🌐 服务商支持

目前主要支持 OpenAI 和硅基流动的余额查询，其他服务商可根据其API文档扩展

  • OpenAI: 通过 /v1/dashboard/billing/subscription 端点查询
  • 硅基流动: 通过 /v1/user/info 端点查询用户信息和余额
  • Anthropic: 预留接口，待官方API支持
  • 其他: 可根据各服务商API文档添加

🛡️ 安全考虑

余额查询是可选的，默认关闭，避免不必要的API调用和潜在的权限问题

📱 用户体验

提供了清晰的UI反馈：

  • 开关状态提示文字
  • 余额信息突出显示
  • 错误处理和降级

使用方法

  1. 在单个Key检测页面输入API Key
  2. 点击"高级设置"展开选项
  3. 开启"查询账户余额"开关
  4. 点击"开始检测"
  5. 在结果中查看余额信息（如果支持）

🎉 功能完成

余额查询功能已完全实现，可以展示真实的账户余额信息！
